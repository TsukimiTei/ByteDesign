Features

Wireframing at the speed of typing

⚡

Instant Generation

Describe any UI component, page, or multi-screen flow in plain language. Get a complete ASCII wireframe in seconds.

🗨

Natural Language Iteration

"Move the sidebar to the right." "Add a search bar to the header." Iterate on designs the way you think — in words.

📱

Web & Mobile

Design for any platform. Desktop dashboards, mobile apps, tablet layouts — just tell ByteDesign what you're building.

📦

Export to JSON

Turn your wireframe into a structured JSON layout description that coding agents can directly consume to build the real thing.

🛠

Built for Claude Code

Runs as a native slash command. No extra apps, no context switching. Design and code in the same terminal session.

🎲

Zero Dependencies

A single markdown file. No runtime, no API keys, no config. Install once, use forever.

How It Works

Three steps. That's it.

1

Install

One command adds ByteDesign to your Claude Code environment.

$ npx bytedesign install
2

Describe

Type /bytedesign followed by what you want to design, in any language.

> /bytedesign an analytics dashboard with sidebar, KPI cards, and a data table
3

Iterate & Ship

Refine the wireframe with follow-up instructions. When you're happy, export JSON and let your coding agent build it.

> Move the nav to the left, make the cards 3 columns instead of 2

Ready to design?

Install ByteDesign and start wireframing in seconds.

npx bytedesign install click to copy

or install globally

npm i -g bytedesign && bytedesign install click to copy
